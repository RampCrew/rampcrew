# 🚤 RampCrew

**North Idaho's Boat Launch Service App**

RampCrew is a mobile-first web app serving Lake Coeur d'Alene, Lake Pend Oreille, and Hayden Lake.

## Features

- **Concierge Launch Help** — Book expert launch crew on demand
- **Ramp Slot Booking** — Reserve your launch time, skip the line
- **Live Ramp Conditions** — Real-time status, wait times, water conditions
- **Helper Marketplace** — Hire verified, ID-checked local helpers
- **Admin Dashboard** — Full ops control: bookings, helpers, ramps, revenue

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- React Router
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

## Coverage Areas

- Lake Coeur d'Alene (8 ramps)
- Lake Pend Oreille (6 ramps)
- Hayden Lake (3 ramps)

## Business Model

White-label licensing to marinas — $2,000/month per location.
